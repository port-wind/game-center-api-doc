游戏中心（Game Center）接入指南

适用对象： 客户端研发、服务端研发、测试人员

更新日期： 2026-03-04
 
文档版本： V1.0.0
1. 前言
本通过标准化接口，游戏端与游戏中心（Game Center，简称 GC）的交互。主要功能包：
获取游戏链接、登录认证、用户信息余额信息的获取、扣减积分、以及游戏记录查询等功能。
接入流程概览：
Game Center会通过游戏的id和用户登陆token向游戏端server请求游戏链接地址。
用户通过游戏链接打开游戏，游戏端server需要向Game Center发起请求验证用户token，并获取用户名和头像信息。
游戏端server请求Game Center获取用户余额信息。
用户玩积分游戏，游戏服务端发起扣分/加分请求到Game Center。
用户退出游戏，游戏服务端向Game Center发送游戏退出请求。
